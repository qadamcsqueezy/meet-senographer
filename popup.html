Meet Stenographer

How to Use

Stenographer and Summarizer for Google Meet transcripts.

  • Join a Google Meet session.
  • The extension automatically records the transcript.
  • After the meeting, a summary is generated using llama 3

Tip: Customize the summary prompt and change API key in Settings.
